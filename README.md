# Classic Snake (JavaScript)

Simple browser-based classic Snake game implemented with HTML, CSS and JavaScript harnessing the power of Github Copilot.

## Features
- Grid-based snake, food and obstacles
- Wrap-around (snake passes through walls)
- Difficulty levels: Novice / Intermediate / Expert (adjusts speed)
- Keyboard controls (arrows / WASD) and difficulty shortcuts (1/2/3)
- No build step â€” runs in any modern browser

## Prerequisites
- macOS / Linux / Windows
- Modern browser (Chrome, Firefox, Safari)
- Optional: Git and a simple HTTP server (python or node) for serving files
